## Команды

Команды для компиляции программ в исполняемый файл:

```bash
gcc a.c -o a.out
gcc d.c -o d.out
gcc e.c -o e.out
gcc h.c -o h.out
```

Команда для создания переменной окружения:

```bash
export MY_VARIABLE=value
```

Команда для вывода значения переменной окружения:

```bash
echo $MY_VARIABLE
```
